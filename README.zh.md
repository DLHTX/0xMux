** tmux 会话管理器，自带 Web UI。**

0xMux 是一个自托管的 Web 应用，让你在浏览器中管理和操作 tmux 会话。通过 WebSocket 实时访问终端，支持桌面和移动端。一个二进制文件，零配置启动。

它把 tmux 变成一个类似 IDE 的浏览器工作区——不用 SSH 手动 attach，打开网页就能看到终端、文件管理、Git 操作等一切功能。

## 安装

```sh
# npx（一次性运行）
npx 0xmux

# npm（全局安装）
npm install -g 0xmux
0xmux

# Homebrew（macOS）
brew install DLHTX/0xmux/0xmux
0xmux
```

打开 `http://localhost:1234` 即可使用。

## 功能

### 终端管理
- 在浏览器中创建、重命名、删除 tmux 会话
- 多窗口支持——每个会话可包含多个窗口，以标签页形式切换
- 分屏——支持水平和垂直分割，和 tmux 操作一致
- 基于 xterm.js 的实时终端渲染，WebGL 硬件加速
- 可拖拽调整大小的分屏工作区

### 文件浏览器和编辑器
- 树形结构浏览文件系统
- 内置代码编辑器，支持语法高亮和 Markdown 预览（Vditor）
- 快速文件搜索（模糊查找）
- 文件上传、重命名、删除、新建——全部在浏览器完成
- 图片预览和内置查看器

### Git 集成
- 查看仓库状态、分支和提交记录
- 暂存/取消暂存文件，查看 diff
- 提交、推送、切换分支、放弃更改
- 所有 Git 操作通过可视化侧边栏完成

### AI 工具管理
- 浏览和安装 AI 编程工具（Claude Code、Cursor 等）
- 管理 MCP 服务器配置
- 跨工具同步全局 AI 配置

### 安全
- 可选的密码认证，使用 Argon2 哈希
- 基于 Token 的会话鉴权中间件
- 首次启动引导向导——设置密码或跳过

### 桌面和浏览器自动化（Agent 模块）
- 远程桌面控制——截图、点击、输入、拖拽
- 浏览器自动化——导航、点击元素、管理标签页
- 定时任务调度器
- UI 无障碍树检查

### 其他
- 移动端适配，虚拟键盘栏和滑动导航
- 应用内通知系统
- 布局持久化——你的面板排列会自动保存和恢复
- 自动更新检测，一键更新
- 依赖检测——如果缺少 tmux 会引导你安装
- 国际化支持（中文 / 英文）
- 单二进制发布——前端通过 rust-embed 嵌入
- 像素风 Brutalist 主题，支持自定义配色

## 技术栈

- **后端:** Rust + Axum + WebSocket + portable-pty
- **前端:** React 19 + TypeScript + Tailwind CSS 4 + xterm.js
- **构建:** 单二进制文件，前端嵌入（rust-embed）
- **分发:** npm + Homebrew + GitHub Releases
