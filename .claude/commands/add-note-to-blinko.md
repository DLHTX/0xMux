# 添加闪念到 Blinko

快速向 Blinko 添加闪念记录，默认添加 `#工作/AI` 标签。

## 功能

使用 Blinko MCP 工具快速记录突发灵感、想法或笔记。

## 主要特性

1. 快速创建闪念（Blinko）或笔记（Note）
2. 自动添加 `#工作/AI` 标签（如果用户未指定其他标签）
3. 支持添加自定义标签（使用 # 开头）
4. 返回创建确认和闪念 ID

## 使用方式

调用此命令时，Claude 会：
1. 接收用户的闪念内容
2. 检查是否包含标签，如果没有则自动添加 `#工作/AI`
3. 使用 `mcp_blinko_upsertBlinko` 工具创建闪念
4. 返回创建成功的确认信息

## 内容类型

- **blinko**（默认）：闪念/突发灵感，适合快速记录想法
- **note**：较长的结构化内容，适合详细笔记
- **todo**：待办任务，需要完成的事项

## 标签规则

- 标签以 `#` 开头，如：`#工作/AI`、`#学习`、`#项目`
- 可以添加多个标签，用空格分隔
- 支持多级标签，使用 `/` 分隔，如：`#工作/AI/机器学习`
- 如果用户内容中没有任何标签，自动添加默认标签 `#工作/AI`

## 工作流程

1. **接收内容**：获取用户想要记录的内容
2. **检查标签**：判断内容是否已包含标签
3. **添加默认标签**：如果没有标签，自动添加 `#工作/AI`
4. **确定类型**：默认为 `blinko`，用户可指定 `note` 或 `todo`
5. **调用 API**：使用 `mcp_blinko_upsertBlinko` 创建记录
6. **返回确认**：显示创建成功的详细信息

## 注意事项

- 闪念内容会立即保存到 Blinko 数据库
- 每条记录都会生成唯一 ID，可用于后续查询或修改
- 支持 Markdown 格式的内容
- 标签有助于后续搜索和分类整理
